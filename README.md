# console-dworp
The greatest debugging tool ever made.

## Usage
`npm install console-dworp`

```javascript
require('console-dworp');

console.dworp();
```

This prints:

`dworp`

Additional arguments can be used as well:

```javascript
console.dworp(42);
```

prints:

`dworp, 42`

`console.blorp`, `console.florp` and `console.derp` are also available.
